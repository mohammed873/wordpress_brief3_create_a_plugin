in order to overview this project you need to :

first download the zip folder of this project or you can pull it using git tool

then rename that folder to "plugin_brief"

secondely once it is downloaded you should move that folder into xaamp/htdocs folder

Upload my_plugin.php to the /wp-content/plugins/ directory

after that you just open a new tap on google and past the following link http://localhost/plugin_brief/wp-admin/plugins.php

after that you find a plugin under the name of "my_plugin"

Activate the plugin through the 'Plugins' menu in WordPress

for viewing the code just head over to plugin_brief\wp-content\plugins\my_plugin
